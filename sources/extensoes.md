# Extensões e Plugins Úteis para Assembly

## Visual Studio Code
- `vscode-asm` — realce de sintaxe para Assembly.
- `x86 and x86_64 Assembly` — suporte a linguagem Assembly e snippets.
- `Native Debug` — depuração nativa para GDB e LLDB.
- `CodeLLDB` — depuração avançada com imagens e variáveis.
- `Better Comments` — organizar notas e lembretes no código.

## Vim / Neovim
- `vim-asmsyntax` — sintaxe Assembly para vários assemblers.
- `YouCompleteMe` / `coc.nvim` — autocompletar e integração com linguagem.
- `vimspector` — depuração interativa dentro do Vim.

## Outros editores e ferramentas
- `Sublime Text` — pacotes de sintaxe Assembly.
- `JetBrains CLion` — suporta projetos C/C++ com inclusão de Assembly.
- `GitHub Codespaces` — ambiente pronto para trabalhar com Assembly.

## Dicas de configuração
- Ative realce de sintaxe para `.asm`, `.s` e `.S`.
- Use snippets para instruções comuns e macros de seção.
- Configure tarefas de build para `nasm` / `gcc` e depuração com `gdb`.
- Mantenha atalhos para abrir `cmd`/`terminal`, compilar e debugar rapidamente.
