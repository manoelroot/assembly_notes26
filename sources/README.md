# Recursos Úteis para Assembly x86_64

Esta pasta reúne materiais de estudo, ferramentas e extensões úteis para aprender e trabalhar com Assembly no Linux.

## O que você encontra aqui
- `aprendizado.md` — cursos, livros, tutoriais e referências.
- `ferramentas.md` — ferramentas de desenvolvimento e análise.
- `extensoes.md` — extensões úteis para editores e IDEs.

## Como usar
1. Abra `aprendizado.md` para escolher os melhores recursos de estudo.
2. Consulte `ferramentas.md` ao preparar seu ambiente de desenvolvimento.
3. Veja `extensoes.md` para configurar seu editor com suporte a Assembly.

## Dicas rápidas
- Combine leitura teórica com prática de código.
- Use ferramentas como `gdb` e `objdump` para entender o comportamento real do código.
- Instale extensões para realce de sintaxe, snippets e integração com debuggers.
