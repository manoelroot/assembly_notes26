# Diagramas de Assembly x86_64

Esta pasta contém diagramas Mermaid para visualizar arquitetura, fluxo de processo e chamadas de sistema.

## O que você encontra aqui
- `architecture-overview.md` — layout de memória do processo e fluxo de syscall.

## Abordagem
Use diagramas para fixar conceitos que são difíceis de visualizar apenas com texto. Eles complementam as explicações em `04-arquitetura/`.

## Como usar
1. Abra `architecture-overview.md` no VS Code.
2. Ative a visualização Mermaid para renderizar os fluxogramas.
3. Compare com `../images/architecture.svg` para ver a mesma ideia em formato de imagem.
4. Use os diagramas como referência rápida durante o estudo.

## Dicas
- Diagramas ajudam a lembrar o fluxo de `syscall` e a estrutura de memória.
- Use-os quando revisar ou ensinar o modelo de execução do Linux.
