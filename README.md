# Assembly Notes x86_64 Linux

Este repositório organiza anotações completas sobre Assembly x86_64 para Linux, com conteúdo teórico, prático, diagramas e recursos de estudo.

## Estrutura do repositório
- `linux_x86_64/` — notas principais de Assembly, sintaxe, arquitetura, exemplos e tópicos avançados.
- `docs/` — regras, tipos, convenções e observações sobre Assembly x86_64.
- `sources/` — recursos de aprendizado, ferramentas e extensões úteis.
- `diagrams/` — diagramas Mermaid para visualizar arquitetura e fluxo de execução.
- `images/` — ilustrações SVG explicativas para apoiar o aprendizado.

## Como usar
1. Comece lendo `linux_x86_64/README.md` para entender a estrutura principal.
2. Consulte `docs/README.md` para regras, convenções e arquitetura de registros.
3. Use `sources/README.md` para encontrar cursos, ferramentas e extensões.
4. Abra `diagrams/` e `images/` para referências visuais que ajudam a fixar os conceitos.

## Fluxo de trabalho Git
- Crie branches temáticos: `feature/<nome>`, `fix/<nome>` ou `docs/<nome>`.
- Use a convenção de commits definida em `COMMIT_GUIDELINES.md`.
- Revise as alterações antes de `git add` e depois faça `git commit` com mensagem clara.
- Faça o push com `git push origin <branch>` e abra pull request se for trabalhar colaborativamente.

## Contribuindo
Leia `COMMIT_GUIDELINES.md` para padronizar mensagens de commit e manter um histórico organizado.
