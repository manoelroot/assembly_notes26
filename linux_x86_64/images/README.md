# Imagens explicativas de Assembly x86_64

Esta pasta contém ilustrações SVG que ajudam a entender visualmente os conceitos de cada seção.

Arquivos:
- `architecture.svg` — layout de memória do processo e conceitos de arquitetura.
- `syntax.svg` — comparação entre sintaxe AT&T e Intel.
- `tools.svg` — fluxo de ferramenta: assembler, linker e debugger.
- `examples.svg` — visão geral dos exemplos práticos.
- `advanced.svg` — panorama dos tópicos avançados.

## Como usar

Abra os arquivos SVG no VS Code ou em um visualizador de imagens. Eles são úteis para revisão rápida, apresentação e estudo visual.
