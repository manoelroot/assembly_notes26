# Instruções x86_64

- Movimentação de dados: `mov`, `lea`
- Aritmética: `add`, `sub`, `mul`, `div`, `inc`, `dec`
- Lógica: `and`, `or`, `xor`, `not`, `test`, `cmp`
- Controle de fluxo: `jmp`, `je`, `jne`, `jg`, `jl`, `call`, `ret`
- Instruções de controle de string e repetição: `rep`, `movsb`, `stosb`
