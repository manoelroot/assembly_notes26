# Introdução ao Assembly x86_64 para Linux

Esta pasta apresenta os fundamentos necessários para começar a escrever Assembly no Linux x86_64.

## O que esta pasta cobre
- Conceitos básicos de Assembly e arquitetura
- Diferenças entre x86 e x86_64
- Modelo de execução de programas
- Fluxo de compilação, montagem e linkagem
- Ferramentas básicas usadas em exemplos

## Abordagem
Comece aqui para entender os blocos básicos e o propósito de cada registrador e seção. Esta pasta prepara você para o conteúdo mais prático e técnico das seções seguintes.

## Como usar
1. Leia `README.md` para a visão geral.
2. Estude `conceitos.md` para detalhes de arquitetura e terminologia.
3. Use `images/architecture.svg` para visualizar o layout do processo.

## Por que isso importa
- Evita erros comuns ao começar em Assembly.
- Mostra como o Linux organiza memória e inicializa um programa.
- Prepara você para escrever código que funcione corretamente no ABI x86_64.
