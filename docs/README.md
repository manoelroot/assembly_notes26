# Regras e Convenções de Assembly x86_64

Esta pasta documenta regras, tipos, convenções e observações para Assembly x86_64 no Linux.

## O que você encontra aqui
- `regras.md` — regras gerais de estilo e boas práticas.
- `tipos.md` — tipos de dados, tamanhos de registradores e operandos.
- `convencoes.md` — convenções de chamada, registro e pilha.
- `observacoes.md` — armadilhas comuns e observações úteis.
- `registradores_arquitetura.md` — explicação completa dos registradores e arquitetura x86_64.

## Abordagem
Esta pasta apresenta critérios claros para código Assembly seguro e legível. Use-a como guia de referência ao escrever, revisar e depurar programas em Assembly.

## Como usar
1. Leia `regras.md` para entender as principais normas de estilo.
2. Use `tipos.md` para consultar tamanhos e representações de dados.
3. Estude `convencoes.md` para escrever chamadas e frames de pilha corretos.
4. Consulte `observacoes.md` para evitar erros comuns.
5. Abra `registradores_arquitetura.md` para entender profundamente o x86_64.
